# piglin-android
猪灵图床 Android端 一键上传图片 管理和分类图片

# 后台API
本APP基于猪灵图床后台API接口实现 该后台接口可实现一键配置 开箱即用

该API基于go+gin+mysql+redis实现

传送门：[https://github.com/tc7326/piglin](https://github.com/tc7326/piglin)
